# topo-op — Soul Resonance Chess

A Soul Eater inspired browser chess game with a dark fantasy theme.

## Run locally

Open `index.html` in a browser, or run a static server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Features

- Full chess movement rules for all pieces
- Castling, en passant, and auto-promotion to queen
- Check, checkmate, and stalemate detection
- Soul Eater inspired UI and piece codex
